MinDoc 安装说明

1、安装方式：
解压安装


2、安装文件：
mindoc-install.sh


3、安装目录
/usr/local/mindoc

/usr/local/mindoc/conf/app.conf 配置文件


4、访问端口
8181


5、默认管理员密码：
admin / 123456


6、数据库(sqlite3)
/usr/local/mindoc/database/mindoc.db



附：操作命令

启动：
systemctl start mindocd

停止：
systemctl stop mindocd

重启：
systemctl restart mindocd
